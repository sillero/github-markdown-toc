GitHub markdown TOC generator (bookmarklet)

Drag THIS LINK to your bookmarks

  1. Visit the target github markdown page, let's say https://github.com/facebook/react/blob/master/README.md
  2. Click the bookmarklet or execute the script
  3. It will output (as an alert and in your browser console) a Table of Contents like this
